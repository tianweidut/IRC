配置如下：
         操作系统:ubuntu (版本号不限)
         运行:解压文件后，在源码所在文件夹中，由终端中执行sh install.sh
         注：可能第一次执行速度较慢，原因是系统安装必备的开发库，推荐选择速度快的ubuntu软件源。
程序说明：
         1.默认扫描了所有端口，不需要再设置
         2.IP地址格式：192.168.7.0/24  （扫描了192.168.7.x网段所有IP）
                       192.168.0.0/16   (扫描了192.168.x.x网段所有IP)
                       192.168.7.1        (扫描单个IP地址)
                       192.168.7.1-25   (扫描由1-25 的 ip地址)
           注：用回车进行区分不同IP格式，一次可以扫描多种地址模式
         3.线程 默认设置32线程，为最大值，范围1-32
         4.程序采用SYN半扫描，绝大部分情况下对方机器不会记录连接信息，而且能穿过部分防火墙。
